Gramática da Lógica Proposicional
Descrição da Gramática

GramLogic é um software que possui uma gramática propria para geração e validação de fórmulas da Lógica Proposicional (ler mais). Logo abaixo é apresentado as regras de sintaxe para a construão da fórmula.

Regras de Construção
Conectivos Negação Letra Predicativas Conclusão Premissas

Os conectivos lógicos abaixo são utilizados pela gramática para representação de formulas proposicionais. Para representar uma fórmula, deve-se utilizar os conectivos listados abaixo.

Operação Conectivos Estrutura Lógica
Negação ~ ~P
Conjunção ^ P^Q
Disjunção v PvQ
Conjunção -> P->Q
Bicondicional <-> P<->Q

A negação pode ser utilizada para negar uma premissa ou uma fórmula relacionadas por conectivos. Neste caso os parênteses são utilizados para organizar a sequência de prioridade da fórmula aplicada. Ou seja, para negar uma fórmula relacionada a conectivo deve utilizar obrigatóriamente parênteses conforme apresentado na tabela abaixo.

Negação de Predicados Negação de Predicados Relacionados por Conectivos
~(P) ~(P->~Q)
~P ~(PvQ)

A gramática utiliza letras maiúsculas podendo ser qualquer letra de "A" a "Z", para representar letras predicativas da fórmula proposicional.

A conclusão é definida através do identificador de conclusão "|-" seguida do argumento de conclusão.

Indicador de Conclusão Argumento de Conclusão
|- ~(P->~Q)

Para definição de premissas(argumentos) é utilizado letras predicativas. Uma fórmula pode ser composta por zero ou mais premissas e obrigatóriamente uma conclusão, para definição de mais premissas deve-se utilizar a vírgula para determinar a separação das premissas.

Premissa Premissas
P->Q P->Q,PvQ
(P->Q) (P->Q),(PvQ)
~Q P->Q,~Q
~(P->Q) ~(P->Q),Q

Sobre

GramLogic foi criado na disciplina de compiladores do curso de Ciência da Computação do Centro Universitário Luterano de Palmas - CEULP\ULBRA, A partir da necessidade de uma estrutura que permitisse a criação e validação de fórmulas da lógica proposicional. É também, parte do projeto Logic Live que está vinculado ao Grupo de Estudos em Novas Tecnologias para processos de Ensino e Aprendizagem (GENTE).

GramLogic possui uma gramática própria criada com auxílio da ferramenta ANTLR, para ler, analizar, processar e criar um documento XML da fórmula estrutural. Visto que as fórmulas utilizadas no projeto LogicLive são extraídas de um documento XML que segue a mesma estrutura gerada pelo software GramLogic.
